## Python Programming Fundamentals

This repository contains exercises and small implementations designed to reinforce core Python programming principles.

### Core Topics Covered
- Variables and data types (strings, integers, floats)
- Control flow (if, elif statements, loops)
- Functions and reusable logic
- Built-in data structures (lists, dictionaries, sets)
- Exception handling by using try–except statements
- Object-Oriented Programming (classes, attributes, methods)

### Learning Outcome
By completing these exercises, I developed a strong understanding of Python syntax, program structure, and OOP usage in logical problem solving, which forms the foundation of backend applications, games and data management systems development.
